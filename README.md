# Akodessewa Supplier Portal

Professional dashboard for auto parts suppliers on Akodessewa.com.

## Features

- **Authentication** - Register, login, password reset
- **Dashboard** - Real-time stats, charts, and quick actions
- **Products** - Full CRUD for auto/moto parts with specs and compatibility
- **Orders** - Accept/reject orders, cancel individual items
- **Used Vehicles** - Publish cars and motos for sale
- **Promotions** - Create percentage or fixed-amount discounts
- **Analytics** - Revenue charts, category breakdowns, export to CSV
- **Wallet** - Track earnings, request withdrawals via Mobile Money
- **Delivery** - Register and manage delivery personnel
- **Settings** - Shop profile, business hours, security

## Tech Stack

- **Next.js 14** (App Router)
- **Tailwind CSS** (utility-first styling)
- **Lucide React** (icons)
- **Recharts** (charts and graphs)
- **TypeScript**

## Getting Started

```bash
cd akodessewa-supplier-portal
npm install
npm run dev
```

Open [http://localhost:3001](http://localhost:3001)

## Color Theme

Uses the same red brand color (`#d32f2f`) as the main Akodessewa website.
